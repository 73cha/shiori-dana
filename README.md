# 栞棚
このアプリは、個人的に利用しているブックマーク管理アプリです。  
Claudeを利用してデザインから実装までを行っています。

## デザイン
ClaudeのMCPでPencilと接続し、プロンプトでやり取りしながら、Vibe Designしています。

## 実装
デザイン同様に、Vibe Codingしています。
