# アプリケーションで発生するイベント
暫定的なイベントを定義。

## グローバルイベント
### initialize
- 初期化時に発生
- 再フェッチで発生

### filter
- タグ検索で発生
- キーワード検索で発生

### update
ブックマークのタグ・タイトルの編集で発生。

### remove
ブックマークの削除で発生

### error
- APIとの通信で発生
- レスポンスステータスが失敗で発生

### success
- APIとの通信で発生
- レスポンスステータスが成功で発生

## ローカルイベント
### reset:pagenation:index
ページネーション内のページ送りに使用しているインデックスをリセットする。
